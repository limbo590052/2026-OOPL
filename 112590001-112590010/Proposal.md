# Game Introduction

植物大戰殭屍 是一款塔防遊戲，玩家需要利用擁有的植物抵抗殭屍，。  
玩家完成每個初期關卡後獲得新的植物，植物可以發射子彈、減速或造成範圍攻擊，而殭屍會從地圖右側慢慢向左移動。  

[遊戲連結](https://youtu.be/XENla8M3910?si=odsjpmxuRFgF3s-h)

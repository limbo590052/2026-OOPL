# Game Introduction

《元氣騎士》是一款超人氣的像素風 Roguelike 彈幕射擊遊戲，由涼屋遊戲在 2017 年開發發行，後來經過不斷的更新與擴充，成為一款廣受歡迎的地牢探索遊戲。遊戲強調直覺的操作與隨機應變，玩家需要閃避滿天飛舞的彈幕，合理搭配各種武器、角色技能與天賦，才能順利通關。玩家扮演不同職業的英雄，為了奪回被外星生物搶走、用以維持世界平衡的魔法石，必須深入危機四伏的地牢。地牢內部充滿了隨機生成的怪物、陷阱、強大首領和寶藏，只有善用自動瞄準、妥善分配能量與資源，突破重重難關。

# Game Introduction

玩家必須在自家草坪的網格上，利用有限的「陽光」資源種植各式各樣具有特殊能力的植物（如發射豌豆的射手、產出資源的向日葵、或擋住敵人的堅果牆），以抵禦一波波進攻、企圖闖入屋內吃掉大腦的殭屍大軍。 

[遊戲連結](https://www.youtube.com/watch?v=XENla8M3910)

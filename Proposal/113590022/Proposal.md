# Game Introduction

本遊戲參考任天堂經典遊戲《坦克大戰 (Battle City)》進行設計。玩家操控坦克在地圖中移動與射擊，消滅場上的敵方坦克。遊戲設計多個關卡，不同關卡會有不同的地形與敵人數量，使難度逐漸提高。玩家被敵方坦克擊中時會失去一條生命，當生命數耗盡時遊戲失敗；若成功擊敗關卡中的所有敵人即可過關。當玩家完成最終關卡後，遊戲將出現結局畫面，代表玩家成功通過所有挑戰並取得勝利。<br>
[遊戲介紹連結](https://youtu.be/umUYIGgivg4?si=zAAGK-ZKhneJmXKZ)

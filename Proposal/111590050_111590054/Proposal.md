# Game Introduction

玩家控制兩個角色，各自擁有不同特性(冰&火)，基本上的邏輯是不能碰到相異顏色的水(熔岩&水)，目標是雙方都進到對應顏色的門，分數是以通關時間計算。
進階目標是要拿到對應的寶石以獲得更高的得分。
目標是做五個關卡。

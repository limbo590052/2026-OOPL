# Game Introduction

This project is a 2D puzzle-platformer game inspired by Fireboy and Watergirl.

The game features two characters: Fireboy and Watergirl.  
The player must control both characters to solve puzzles and reach the exit door.

Each character has different abilities and limitations:
- Fireboy can walk through lava but cannot touch water.
- Watergirl can walk through water but cannot touch lava.

Players must cooperate with both characters to activate switches, move platforms, and avoid hazards.  
The goal of the game is to guide both characters safely through each level and reach their respective exits.

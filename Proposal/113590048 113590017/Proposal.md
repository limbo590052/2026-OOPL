# Game Introduction

《西瓜遊戲》是一款風靡全球的物理益智休閒遊戲，最初因其簡單直覺的操作與充滿不確定性的物理碰撞而爆紅。
當兩個相同的水果接觸時，它們會觸發「演化」機制，合成體積更大的下一階水果。
玩家的最終目標是透過精妙的佈局與連鎖反應，在水果溢出邊界前合成出最高等級的「大西瓜」。


[遊戲連結](https://www.youtube.com/watch?v=BDqJUJ-jOSQ)

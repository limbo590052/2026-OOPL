# Game Introduction

這是一款 2D 惡意陷阱平台跳躍遊戲，玩法靈感來自 Cat Mario 類型作品。玩家需要操作角色在地圖中移動、跳躍、躲避各種陷阱，並抵達終點過關。地圖中除了常見的平台、尖刺與掉落陷阱外，也會加入按鈕與機關系統，例如踩下按鈕後開門、生成平台、觸發障礙物移動或改變地圖路線。玩家需要觀察地圖、記住陷阱與按鈕效果，透過多次嘗試找出正確的過關方法。  
本專案將使用 OOP 方式設計角色、陷阱、平台、按鈕、機關、關卡與 UI 等類別，並透過封裝、繼承、多型來提升程式架構的清晰度與擴充性。

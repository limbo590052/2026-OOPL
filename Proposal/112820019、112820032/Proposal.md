# Game Introduction

《骰子阿瓦隆》是一款在骰子構築策略遊戲。玩家透過投擲與收集特殊骰子，組合點數來擊敗敵人。遊戲核心在於平衡風險與獎勵，類似 Farkle 規則，投擲次數越多力量越強，但也可能因失敗（Farkle）失去所有積累。

預期完成內容如下：

- 會死掉：玩家角色有生命值，遭受敵人攻擊傷害歸零時會進入死亡畫面並可重新挑戰。
- 會獲勝：每個戰鬥關卡會有一名敵人，使其生命值歸零即可進入下一關，完成最終關卡擊敗boss後進入勝利畫面。
- 有關卡：至少規劃多個關卡（含一般關、商店關與 Boss 關），難度隨進度提升。

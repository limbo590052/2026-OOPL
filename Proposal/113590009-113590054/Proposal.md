# Game Introduction

本作是致敬的 2D 橫向捲軸動作遊戲。開發團隊致力於在現代網頁環境下，重現《超級瑪利歐
》最核心的「跳躍回饋感」與「關卡探索樂趣」。玩家將扮演主角，在充滿機關與敵人的蘑菇王
國中穿梭，透過精準的平台跳躍與策略性的道具使用，最終突破重圍抵達終點旗桿。

[遊戲連結](https://www.youtube.com/watch?v=g3aN7P2wMOM)

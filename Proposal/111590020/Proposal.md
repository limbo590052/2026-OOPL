# Game Introduction

瑪麗貓是一款 2D 橫向卷軸遊戲，玩家必須要通過陷阱、怪物、障礙通過8個關卡，到達最終終點。  
每一個關卡的元素以及地圖會有差異，需要透過玩家不斷嘗試才能夠讓角色活著通過。

[遊戲連結](https://youtu.be/ryy3oBlrMvY?si=6uTz0Ag1EbpNhi4-)

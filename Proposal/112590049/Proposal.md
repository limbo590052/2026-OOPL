# Game Introduction

元氣騎士（Soul Knight）是一款以地牢探險為主題的 2D 俯視角 Roguelike 動作射擊遊戲。玩家將扮演不同的英雄在地牢中不斷深入，收集各種奇特的武器，並擊敗充滿隨機性的怪物與強大的 Boss。
遊戲特色包括隨機生成的地牢房間、流暢的雙搖桿（或鍵鼠）射擊體驗、以及多樣化的武器與機制。玩家需要在密集的彈幕中靈活走位穿梭，並妥善利用能量、護甲與不同武器的特性來突破重重關卡。

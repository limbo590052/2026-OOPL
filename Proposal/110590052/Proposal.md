# Game Introduction

火柴人守城 4 (eight defender's) 是一款塔防類遊戲，玩家需要透過擊殺敵人獲取資源、升級火柴人並進行戰略性的職業轉職來抵禦不斷襲來的敵人。  

[遊戲連結](https://web.archive.org/web/20201101125158/http://www5f.biglobe.ne.jp/~pasting/game/EightDefenders_.html)

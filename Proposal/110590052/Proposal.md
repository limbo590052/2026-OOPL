
# 遊戲名稱：火柴人守城4 eight defender's

組員：

110590052 魏名宏

核心機制概覽
防守方屬性：單位等同於「防禦塔」，不具備生命值，不會被敵方攻擊。僅有：攻擊狀態、休息（冷卻）狀態。

資源系統：

金錢 (Gold)、經驗 (EXP)：用於基礎等級提升、用於職業突破與轉職。

職業架構 (1+2+3/2)：

職業 A (戰士系)：初級(1) → 中級(2) → 上級(3)。

職業 B (弓手系)：初級(1) → 中級(2) → 上級(3)。

職業 C (盜賊系)：初級(1) → 中級(2) → 上級(2)。

職業 A 與 職業 B 的其中一個上級職業為「相同職業」

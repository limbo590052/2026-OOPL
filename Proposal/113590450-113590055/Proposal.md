# Game Introduction

《小精靈》（Pac-Man）是一款經典街機迷宮遊戲，最早由 Namco 推出。玩家操控主角「小精靈」在迷宮中移動，目標是吃掉地圖中的豆子、避開鬼怪追擊，並成功完成每一關卡。
玩家控制小精靈在迷宮中移動，吃掉地圖上的所有豆子即可過關。
在遊戲過程中，地圖內會有鬼怪不斷追逐玩家，一旦被鬼碰到就會失去生命。
部分版本中還有「能量豆」，吃下後小精靈可以在短時間內反過來吃掉鬼怪，增加遊戲的變化性與趣味性。
